## Organização do repositório

A estrutura do repositório segue uma organização por temas. Os assuntos iniciais, que correspondem à base da linguagem, estão localizados na pasta **_Básico_**. Nela, você encontrará arquivos de texto com explicações teóricas e exemplos de problemas resolvidos por mim.

Na pasta definida como **_Intermediário_**, você irá encontrar o básico da teoria de _Programação orientada a objeto_ (POO), alguns projetos onde temos trabalhos com arquivos, tratamento de banco de dados, principalmente em arquivos csv, e contrução de banco de dados mais limpo, com algumas observações, alguns desafios em python (importantes para gravar a teoria) e estudos com as bibliotecas que eu acredito serem as principais para coleta de informação a partir dos dados, **_Pandas, Numpy, Seaborn, Matplotlib_**.

Na pasta **_Avançada_** você vai ter os conceitos de _Machine Learning_ e _Deep Learning_ e terá uma nota extra de Algoritimo Genético, que é muito utilizado para modelos onde as funções não são diferenciaveis (Sim, no final é tudo matemática)

Na pasta **_DataClear_** tem alguns projetos que estou utilizando como base para testar os conceitos que eu vou aprendendo, pois lá tem 3 banco de dados diferentes. 

Algo que vou **tentar* fazer durante a apresentação dos códigos é apresentar conceitos matemáticos muito pesados, principalmente para o conteúdo avançado, o motivo é que talvez nem todos possuam a base de cálculo diferêncial e Integral mas aconselho fortemente a tentar ler sobre ou estudar um pouco por fora, principalmente levando em consideração os avanços tecnológicos e de IA.

Não vou entrar no assunto de computação quântica, acredito que isso já foge de qualquer arcabouço teórico mas posso deixar na bibliografia algumas referências caso se interesse.

## Quem Sou eu?

Meu nome é Anderson Lima dos Santos formado em física na Universidade de São Paulo. Atuo há aproximadamente quatro anos com análise de dados científicos, com foco na pesquisa em farmacos, além de desenvolver simulações computacionais aplicadas a sistemas biológicos.

Ao longo da minha trajetória, trabalhei com diferentes sistemas biológicos, incluindo o vírus da COVID-19, durante um período de três anos, e com protozoário Trypanosoma cruzi e sua proteína cruzaína, por cerca de dois anos. Nesse intervalo, executei simulações de mecânica clássica e mecânica quântica, que resultaram na produção de um artigo científico atualmente em fase final de elaboração.

Minhas atividades envolviam a execução de simulações em ambientes computacionais, bem como a análise dos dados gerados, tais como energia mínima, aminoácidos em contato, ligações químicas, regiões de interação, entre outros fatores biológicos, tanto numéricos quanto visuais. Meu trabalho também girava em automatizar processos longos de simulação de maneira que o usuário não precisasse ter contato direto a todo momento consiliando diferentes sistemas e softwares computacionais. Essas análises tiveram como objetivo compreender e explicar o funcionamento de medicamentos atualmente testados para o tratamento dessas doenças.

O objetivo deste repositório é apresentar o Python como uma das principais ferramentas utilizadas por mim para a escrita de scripts voltados à análise de dados científicos, bem como demonstrar ferramentas gráficas e bibliotecas úteis para esse tipo de análise.

Por fim, este repositório tem como finalidade compor meu portfólio profissional, servindo futuramente como uma demonstração prática de conhecimentos técnicos para processos seletivos e oportunidades em grandes empresas.
